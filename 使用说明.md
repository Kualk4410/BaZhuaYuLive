# 前端部署

- front 目录下运行 npm install
- front/server 目录下运行 npm install
- front/render 目录下运行 npm run dev

# 后端使用

- 安装 back 目录下的依赖库 pip install -r requirements.txt
- 启动后端接口 `cd ./back && python -m flask.v1.app` （以 back 为根目录启动）

- 配置后端参数 back/config.yml（参数包括以下内容）

```yaml
module: deepseek-v3-250324
base_url: https://ark.cn-beijing.volces.com/api/v3
api_key: xxx
xlsx_path: xxx
```

# 网络数据传输规范

## 响应格式

```json
{
  "code": 0,
  "msg": ""
  "data": Object,
},
```

- `code` 响应码，规定 1 为成功，0 为失败
- `msg` 响应消息，自然语言，方便理解；失败则是失败的原因
- `data` 返回的数据，成功了就会响应数据（格式参照接口），失败则为空
